# 🎬 MovieVerse – Movie Explorer Web App

## 📌 Project Overview

MovieVerse is a modern movie explorer web application built using React and JavaScript. The application allows users to discover movies, search for their favorite titles, and explore detailed information such as ratings, release dates, and posters.

This project demonstrates the use of API integration, data manipulation using JavaScript, and responsive UI design.

---

## 🚀 Features

* 🔍 **Search Movies** – Search for any movie by title
* 🎭 **Filter Movies** – Filter movies based on genres
* ⭐ **Sort Movies** – Sort by rating, popularity, or release year
* 📄 **Movie Details** – View detailed information about each movie
* 🌙 **Dark/Light Mode** *(optional)*
* ❤️ **Favorites System** *(optional – using localStorage)*

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Languages:** JavaScript, HTML, CSS
* **API:** TMDb (The Movie Database API)
* **State Management:** React Hooks (useState, useEffect)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/movieverse.git
```

2. Navigate to project folder:

```bash
cd movieverse
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
